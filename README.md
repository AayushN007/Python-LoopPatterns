# Python Loop Patterns

## 📘 Overview

**Python Loop Patterns** is a collection of beginner-friendly Python programs that demonstrate the use of **`for` loops**, conditional statements, and arithmetic operations to generate different numerical sequences and patterns.

These programs help build a strong foundation in iteration, sequence generation, and mathematical logic using Python.

## 📂 Programs Included

### 🔹 1. Reverse Number Sequence

Generates numbers from **n to 1** in reverse order.

**Example**

```text
Input: 5
Output: 5 4 3 2 1
```

---

### 🔹 2. Cube Number Generator

Generates the first **n cube numbers**.

**Example**

```text
Input: 5
Output: 1 8 27 64 125
```

---

### 🔹 3. Powers of 3 Generator

Generates the first **n powers of 3**.

**Example**

```text
Input: 6
Output: 1 3 9 27 81 243
```

---

### 🔹 4. Alternating Positive and Negative Sequence

Generates the sequence:

```text
1 -2 3 -4 5 -6 ...
```

---

### 🔹 5. Numbers Divisible by 3 and 5

Displays all numbers from **1 to n** that are divisible by **both 3 and 5**.

**Example**

```text
Input: 50
Output: 15 30 45
```

---

### 🔹 6. Triangular Number Sequence

Generates the sequence:

```text
1 3 6 10 15 21 ...
```

using cumulative sums.
